# Peklo Upeklo!

> 2D puzzle-strategy game with a planning and a routing phase.
> Grid-based level construction using tracks and processing station tiles.
> Custom level system.
> Dynamic audio engine.
> Procedural sprites, animations, and particles.

Project for [Computer Games](https://bilakniha.cvut.cz/en/predmet4699006.html) course, written in Godot Engine in 2020 by four people.
I was the team leader and we won [the best game award](https://cent.felk.cvut.cz/courses/39HRY/archives/2022-2023/) of that year's class.
You can [try the project](https://cent.felk.cvut.cz/courses/39HRY/archives/2022-2023/#team10) in your browser.
